# Collections in Java

This repository demonstrates the usage of Java Collections Framework, focusing on `HashMap` and `HashSet`, along with a revision folder for consolidating knowledge.

---

## Folder Structure and Descriptions

### 1. **HashMap**
   - Contains examples of working with `HashMap` in Java.
   - **Files**:
     - `HashMapExample.java`: Demonstrates creating, updating, and retrieving key-value pairs using `HashMap`.
     - `README.md`: Documentation for `HashMap` usage.

### 2. **HashSet**
   - Contains examples of working with `HashSet` in Java.
   - **Files**:
     - `HashSetExample.java`: Demonstrates adding, removing, and iterating over elements in a `HashSet`.
     - `README.md`: Documentation for `HashSet` usage.

### 3. **Revision**
   - A consolidated example for revisiting key concepts related to Java collections.
   - **Files**:
     - `Main.java`: A comprehensive example that combines `ArrayList`, `HashMap`, `HashSet`, or other concepts for review.
     - `README.md`: Documentation for the revision folder.

---

## Topics Covered

- **HashMap**:
  - Key-value pair storage and retrieval.
  - Common methods like `put`, `get`, `remove`, and iteration.
- **HashSet**:
  - Storing unique elements.
  - Operations like `add`, `remove`, and `contains`.
- **Revision**:
  - Integrates concepts for practical implementation and review.
